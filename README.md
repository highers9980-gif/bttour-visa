# BT TOUR Visa Management System

Vietnamese-First Visa Application Management.

Quan ly don xin visa tu dong cho cac ung vien tu Viet Nam sang Han Quoc.

## Features

- Multi-language: Tieng Viet, English, Korean
- Mobile First: iPhone, Android optimized
- Real-time Dashboard: Google Drive mirroring
- Complete Documents: 20+ document types
- Auto Processing: Gemini OCR workflow
- Auto PDF: application packet generation after completion

## Deploy

Live: https://bttour-visa.vercel.app

## Usage

1. Prepare files with names like `passport_NguyenVanA_S0123.pdf`.
2. Upload files to the BT TOUR Google Drive folder.
3. Open the website and run processing.
4. Review generated application documents.
